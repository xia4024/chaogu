页面路由

GET / → scan_list.html (扫描记录列表，按日期倒序)
GET /scan/<id> → scan_detail.html (单次扫描详情 + 交易计划表格)
GET /holdings → holdings.html (持仓列表 + 新增/编辑表单)
POST /holdings/new → holdings.html (新增持仓)
POST /holdings/<id>/edit → 重定向 (编辑持仓)
POST /holdings/<id>/delete → 重定向 (删除持仓)

启停命令

# 启动 Web 服务（默认 5000 端口）
python web.py

# 指定端口
python web.py --port 8080

# 定时扫描（不变，新增 MySQL 持久化）
python run_scheduled.py --config config.toml --data-source auto --send-email
